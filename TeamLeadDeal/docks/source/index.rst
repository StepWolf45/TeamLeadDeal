Что мы использовали?
----------------------------------------
Как уже было сказано, сайт написан на языке ``Python`` при помощи фреймворка ``Django``. Весь список библиотек
можете увидеть в файле requirements.txt на нашем `GitHub`_. Чтобы скачать все нужные модули используйте команду:

.. code-block::

   pip install -r requirements.txt
